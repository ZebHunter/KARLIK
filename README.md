# KARLIK (Кармический рогалик)
## Описание
Survivors-like игра. Основная идея - выживание в течение времени, заданном на сгенерированной карте. Однако ключевая механика - при убийстве врагов остальные враги эволюционируют и игра усложняется. Отсюда появляется баланс сложности - аккуратный геймплей с большей выживаемостью, либо получение апгрейдов, но сильное усложнение игры. Основной цикл игры - игрок появляется на карте и должен выжить в течение некоторого количества времени. Во время игры появляются волны врагов, которые эволюционируют при их убийстве до следующего уровня. На уровне генерируются склепы, наносящие урон и разрушающиеся при повышении уровней врагов.
## Функционал/Недостатки
- Система кармы - основная механика, игра зависит от уровня кармы. Система влияет на геймплей и состояние карты.
- Система опыта - при убийстве врагов появляются очки опыта, которые при достижении определенного уровня повышают характеристики игрока.
- Активные эффекты - на сгенерированной карте появляются активные эффекты случайным образом, восстанавливающие здоровье или повышающие урон.
- Эволюция врагов - при снижении уровня кармы до определенного порога враги на карте меняют свой тип на следующий в эволюционной цепи, что усложняет геймплей
- Генерация карты - карта состоит из блоков 16x16, в которых генерируются случайным образом объекты - непроходимые (камни), разрушаемые (склепы) и проходимые.
- Механика склепов - на карте есть объекты, наносящие урон при приближении. Их можно разрушить с помощью атаки. При эволюции врагов некоторые случайно разрушаются, добавляя число активных врагов на карту. При достижении низшего уровня кармы все склепы разрушаются.
- Геймплейный цикл по таймеру - для победы требуется выжить в течении заданного времени (2-5 минут).
- Боевая система - автоатака в ближнем бою.
- Передвижение - игрок двигается в любом направлении, которыми также конфигурируется направление атаки. Враги всегда движутся в сторону героя и при приближении начинают атаковать.
Недостатки:

- Звуковое сопровождение отсутствует
- Карта ограничена в генерации статичными объектами
- Отсутствуют некоторые анимации (например смерти врага, разрушения крипты)
- Отсутствие отображения текущего уровня
